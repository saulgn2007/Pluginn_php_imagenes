# Pluginn_php_imagenes

## 🔍Objetivos🔎 

El objetivo es desarrollar un plugin en PHP para el redimensionamiento de imagenes. Todo ha sido desarrollado con la herramienta Antigravity.

## 🎯Requisitos

- Un apartado para el redimensionamiento de imagenes a tamaño logo (Tamaño logo por defecto de WordPress)
- Un apartado para el redimensionamiento de imagenes a tamaño banner (Tamaño banner por defecto de WordPress)
    

## 💻Tecnologias

La tecnologia usada es la siguiente:
- Para el desarrollo del plugin he usado como herramienta principal antigravity con su propio editor de codigo, ya que es el que han solicitado.

## 🖥️¿Como ejecutarlo?
Para ejecutarlo, debemos tener un seervidor local instalado e iniciado. En mi caso con la aplicaccion XAMP he iniciado el servidor apache y en el navegador debemos buscar la URL local del proyecto.
